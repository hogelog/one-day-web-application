# 1 Day Webアプリケーション

## セットアップ

この演習ではRubyとRuby on Rails、PostgreSQLがインストールされている必要があります。

以下のコマンドを実行しインストールされていることを確認してください。

```sh
$ ruby -v
ruby 2.6.5p114
$ rails -v
Rails 6.0.2.1
$ psql --version
psql (PostgreSQL) 12.1
```
